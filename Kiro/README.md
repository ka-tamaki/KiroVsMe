# CSV Graph Reporter

CSVデータから報告書用のグラフを作成するデスクトップアプリケーション

## 機能

- ✅ CSVファイルのドラッグ&ドロップ対応
- ✅ データプレビュー表示
- ✅ グラフ用の列選択機能
- ✅ データ計算式の適用（例: A列*2+180）
- ✅ グラフタイトル・軸ラベルの編集
- ✅ 折れ線グラフ・棒グラフの切り替え
- ✅ グラフのクリップボードコピー
- ✅ Excelファイルへの出力
- ✅ モダンなUI

## セットアップ

```bash
npm install
```

## 開発

```bash
npm run dev
```

## ビルド

```bash
npm run build
```

ビルド後、`dist`フォルダに実行可能ファイルが生成されます。

## 使い方

1. CSVファイルをドラッグ&ドロップまたは選択
2. データプレビューで内容を確認
3. X軸・Y軸に使用する列を選択
4. 必要に応じて計算式を入力（例: 列名*2+180）
5. グラフタイトルや軸ラベルを編集
6. グラフをコピーまたはExcelに出力
